Acerca de mí

Soy Enrique Alejandro García Mota, 23 años, Full Stack Developer en Xeleva Group. Ingeniero en Entornos Virtuales y Negocios Digitales. Especializado en Go, MongoDB, React y TypeScript.

¿Quién soy como desarrollador?

Soy Full Stack Developer en Xeleva Group, donde construyo sistemas backend escalables y aplicaciones web modernas. Mi expertise principal está en Go para servicios backend de alto rendimiento, combinado con MongoDB para gestión flexible de datos. Me especializo en programación concurrente, arquitectura de microservicios y diseño de sistemas que manejan carga real de manera eficiente.

Backend

  • Go (Golang) - Servicios de alto rendimiento con patrones de concurrencia
  • MongoDB - Diseño de bases de datos NoSQL escalables
  • gRPC - Comunicación eficiente entre microservicios
  • WebSockets - Funcionalidades en tiempo real
  • Unit Testing - Confiabilidad del código

Frontend

  • React + TypeScript - Interfaces type-safe y mantenibles
  • State Management - Redux, Zustand, Context API
  • React Router - Navegación compleja
  • Next.js - Server-side rendering y SEO
  • React Testing Library - Testing de UX

Testing & DevOps

  • Cypress, Playwright - Automatización end-to-end
  • CI/CD - Integración y despliegue continuo
  • Cloud Deployment - Gestión en la nube
  • Python - Automatización e IA

Mi Enfoque Técnico

Trabajo con el ciclo completo de desarrollo, desde testing automatizado hasta optimización de rendimiento y despliegue. Mi enfoque técnico se centra en clean architecture, escalabilidad y código production-ready que crece con las necesidades del negocio.

Utilizo patrones de concurrencia en Go, aprovechando goroutines y channels para construir servicios responsivos y eficientes que manejan miles de peticiones concurrentes.

Más Allá del Código

Como Ingeniero en Entornos Virtuales y Negocios Digitales, tengo una perspectiva estratégica sobre cómo la tecnología genera valor. No solo entiendo cómo construir sistemas, sino por qué importan y cómo encajan en objetivos empresariales más amplios.

También dirijo Celinki, un negocio en crecimiento enfocado en venta y reparación de accesorios para celulares. Gestionar Celinki ha afilado mis habilidades en marketing digital, gestión de operaciones, experiencia del cliente y planeación financiera.

Mi Filosofía

Las mejores soluciones surgen de entender tanto el lado técnico como el humano de un problema. No me interesa construir cosas solo porque están de moda—quiero crear herramientas y experiencias que resuelvan problemas reales, escalen inteligentemente y respeten a las personas que las usan.

Equilibrio y Disciplina

Como esposo y padre de un hijo de 5 años, balancear la vida familiar con el crecimiento profesional me ha hecho altamente disciplinado e intencional sobre cómo uso mi tiempo. Valoro la eficiencia, claridad y propósito en todo lo que hago.
